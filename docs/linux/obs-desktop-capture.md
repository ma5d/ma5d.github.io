# 利用虚拟摄像头完成多屏幕共享

 

> 疫情期间，腾讯会议，钉钉等等软件成为人们交流的主要途径，便会产生想要共享多个屏幕的需求，以下采用虚拟摄像头为你提供一个解决方案。

所需软件：OBS-Studio、OBS-VirtualCam插件
